## 🎯 任务功能说明

### 1️⃣ 每日健康检查
**执行时间:** 每天 08:00  
**功能:**
- 获取BTC实时价格和波动率
- 检查交易系统运行状态
- 运行错误日志统计
- 总结今日重要事项

### 2️⃣ 每日记忆整理
**执行时间:** 每天 22:00  
**功能:**
- 回顾今日对话要点
- 提取重要决策到MEMORY.md
- 清理临时笔记文件
- 更新memory索引

### 3️⃣ 每周错误统计
**执行时间:** 每周一 09:00  
**功能:**
- 运行 `error_logger.py --stats`
- 分析本周新错误模式
- 更新patterns.md
- 生成改进建议

### 4️⃣ 每周技能回顾
**执行时间:** 每周日 20:00  
**功能:**
- 回顾本周使用的Skills
- 评估每个Skill的效果
- 识别需要改进的技能
- 更新Skill配置或学习新技能

### 5️⃣ 每月进化回顾
**执行时间:** 每月1日 10:00  
**功能:**
- 量化本月成长指标
- 识别重大改进和突破
- 设定下月进化目标
- 更新MEMORY.md核心记录

### 6️⃣ 每月系统优化
**执行时间:** 每月1日 11:00  
**功能:**
- 分析资源使用情况
- 优化OpenClaw配置
- 清理无用文件
- 更新文档和说明

## 📊 任务管理命令

### 查看所有任务
```bash
openclaw cron list
```

### 查看任务详情
```bash
openclaw cron list --json
```

### 手动运行任务
```bash
openclaw cron run <任务ID>
```

### 查看任务历史
```bash
openclaw cron runs <任务ID>
```

### 禁用任务
```bash
openclaw cron disable <任务ID>
```

### 启用任务
```bash
openclaw cron enable <任务ID>
```

### 删除任务
```bash
openclaw cron rm <任务ID>
```

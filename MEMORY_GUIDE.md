# 📚 记忆系统使用指南

*优化后的双层记忆结构*

---

## 🎯 设计理念

```
短期记忆 → 每日记录（memory/*.md）
     ↓ 自动整理
长期记忆 → MEMORY.md（核心知识）
     ↓ 每周回顾
进化成长 → 量化指标追踪
```

---

## 📁 文件结构

```
~/.openclaw/workspace/
├── MEMORY.md                    ← 长期记忆（核心知识库）
├── MEMORY_OPTIMIZED.md           ← 优化版索引模板（v2.0）
├── MEMORY_GUIDE.md              ← 本使用指南
├── CRON_SYSTEM.md               ← 定时任务说明
├── error-logs/                  ← 错误日志系统
└── memory/
    ├── README.md               ← 每日记录指南
    ├── 2026-02.md             ← 2月月度索引
    ├── 2026-02-07.md          ← 每日记录（示例）
    └── YYYY-MM-DD.md           ← 每日记录（自动创建）
```

---

## 🚀 快速上手

### 1️⃣ 记录每日见闻
```bash
# 创建/编辑今日记录
nano ~/.openclaw/workspace/memory/$(date +%Y-%m-%d).md
```

### 2️⃣ 更新核心知识
```bash
# 编辑MEMORY.md
nano ~/.openclaw/workspace/MEMORY.md
```

### 3️⃣ 记录错误
```bash
# 快速记录错误
~/.openclaw/workspace/error-logs/quick-log-error.sh '错误描述' technical medium
```

### 4️⃣ 查看统计
```bash
# 错误统计
python3 ~/.openclaw/workspace/error-logs/error_logger.py --stats
```

## 📝 每日记录模板

```markdown
# YYYY-MM-DD

## 📝 今日要点
- 与用户的对话主题
- 重要决策或发现

## 💡 学到的新知
- 新技能/新工具
- 新知识/新见解

## 🔧 完成的任务
- 项目进展
- 系统改进

## ⚠️ 遇到的问题
- 错误（可能需要记录到error-logs）
- 困难（需要帮助）

## 📅 明日计划
- 待办事项
- 学习目标
```

---

## 🏷️ 标签系统

### 推荐标签
- `#重要` - 需要长期记忆
- `#BTC` - 交易相关内容
- `#AI` - AI/ML相关
- `#工具` - 技能/工具相关
- `#错误` - 需要追踪的问题
- `#计划` - 待办事项

### 使用示例
```markdown
- #BTC 交易系统更新，准确率提升到80.4%
- #错误 API调用失败，已记录到error-logs
- #重要 用户偏好中文沟通
```

## 🔄 自动化流程

### 每日流程（自动）
1. 08:00 → 定时任务：健康检查
2. 每日 → 自动创建每日记录
3. 22:00 → 定时任务：记忆整理

### 每周流程（自动）
1. 周一 → 错误统计
2. 周日 → 技能回顾

### 每月流程（自动）
1. 每月1日 → 进化回顾
2. 每月1日 → 系统优化

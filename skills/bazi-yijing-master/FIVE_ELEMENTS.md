## 五行生克关系

### 相生（促进、滋养）
```
木 → 火 → 土 → 金 → 水 → 木
```

**详解：**
- 木生火：木柴燃烧产生火
- 火生火：燃烧产生灰烬（土）
- 土生金：金属矿石从土中提炼
- 金生水：金属容器能盛水
- 水生木：水分滋养树木

### 相克（制约、克制）
```
木 → 土 → 水 → 火 → 金 → 木
```

**详解：**
- 木克土：树根穿透土壤
- 土克水：土堤阻挡水流
- 水克火：水能灭火
- 火克金：火能熔化金属
- 金克木：斧头砍伐树木

### 生克辩证
```
相生不是越生越好 → 适度
相克不是越克越好 → 平衡
过犹不及 → 中和
```

分析：

## 常见问题

### Q1：五行缺什么就补什么？
**A**：不是。缺不代表需要补，要看喜忌。

### Q2：名字要按五行补吗？
**A**：参考用神，但不用强求。

### Q3：水晶、宝石有用吗？
**A**：有一定心理作用，主要还是调整心态。

### Q4：风水调整有多大作用？
**A**：有辅助作用，但不能替代自身努力。

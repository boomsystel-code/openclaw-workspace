## 💡 核心知识点总结

### 五行速记

**五行特性口诀：**
```
木：生长、条达 → 仁 → 肝胆 → 春 → 东
火：炎热、向上 → 礼 → 心小肠 → 夏 → 南
土：生化、承载 → 信 → 脾胃 → 四季 → 中
金：清洁、收敛 → 义 → 肺大肠 → 秋 → 西
水：寒冷、向下 → 智 → 肾膀胱 → 冬 → 北
```

**相生口诀：**
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
```

**相克口诀：**
```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```

### 排盘核心口诀

**五虎遁年起月干：**
```
甲己之年丙作首
乙庚之年戊为头
丙辛必定寻庚起
丁壬壬位顺行流
戊癸之年何方发
甲寅之上好追求
```

**五鼠遁日起时干：**
```
甲己还生甲
乙庚丙作初
丙辛从戊起
丁壬庚子居
戊癸何方发
壬子是真途
```

## 🎯 今日重点掌握

### 1. 日主强弱判断公式
```
身强 = 得令(40%) + 得地(30%) + 得势(30%) ≥ 60%
身弱 = 失令 + 失地 + 失势 < 60%
```

### 2. 用神选取规则
```
身强 → 用官杀、食伤、财星（克泄耗）
身弱 → 用印星、比劫（生扶）
```

### 3. 重要概念
```
得令：日主五行与月支五行相同
得地：日主在地支有根
得势：印星、比劫多寡
```

## ❓ 今日疑问

Q1：身强身弱的判断标准是多少？
A1：综合得令、得地、得势三者权重。一般60分以上为身强，40分以下为身弱，中间为偏弱偏强。

Q2：五行缺水就补水吗？
A2：不是。要看日主需要，缺不代表需要补。如果日主本身水旺，再补水反而有害。

Q3：十神中的正偏如何区分？
A3：同阴阳为正（正官、正财、正印），不同阴阳为偏（七杀、偏财、偏印）。

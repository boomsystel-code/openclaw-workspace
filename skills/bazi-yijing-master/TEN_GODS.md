# 十神体系详解

## 十神基础

### 什么是十神
```
以日干为中心
根据五行生克关系
确定其他七个字的名称
```

### 十神口诀
```
我生者 → 食神、伤官
生我者 → 正印、偏印
克我者 → 正官、七杀
我克者 → 正财、偏财
同我者 → 比肩、劫财
```

## 十神详解

### 1. 比肩

**定义**：与日干同五行、同阴阳

**象征**：
```
同辈、同学、朋友
竞争对手
自己本身
```

**性格特征**：
```
优点：独立、自主、竞争意识强
缺点：固执、好胜、人缘较差
```

**身旺时**：
```
适合独立创业
不宜与人合作
工作易变动
```

**身弱时**：
```
需要朋友帮助
适合团队合作
贵人运好
```

### 2. 劫财

**定义**：与日干同五行、不同阴阳

**象征**：
```
兄弟姐妹
争夺财富者
小人、竞争对手
```

**性格特征**：
```
优点：机敏、应变能力强
缺点：投机、好赌、财务易损
```

**特点**：
```
劫财克正财 → 克婚姻、克财
八字有劫财 → 婚姻需特别注意
```

### 3. 食神

**定义**：日干所生、同阴阳

**象征**：
```
子女、晚辈
作品、才华
思想、表达
```

**性格特征**：
```
优点：聪明、才华、温和
缺点：懒散、缺乏斗志
```

**人生方面**：
```
食神生财 → 财源广进
食神制杀 → 化杀为权
食神旺 → 适合艺术、创作
```

### 4. 伤官

**定义**：日干所生、不同阴阳

**象征**：
```
晚辈、学生
才华、技艺
名声、官非
```

**性格特征**：
```
优点：聪明、创新、口才
缺点：叛逆、不服管、爱表现
```

**特点**：
```
伤官见官 → 麻烦不断
伤官配印 → 贵格
伤官旺 → 适合自由职业
```

### 5. 正财

**定义**：日干所克、同阴阳

**象征**：
```
妻子（正财为正妻）
正当收入
固定资产
```

**性格特征**：
```
稳重、务实、理财好
保守、缺乏魄力
```

**特点**：
```
正财旺 → 收入稳定
正财克正印 → 财克印
身弱财多 → 富屋贫人
```

### 6. 偏财

**定义**：日干所克、不同阴阳

**象征**：
```
父亲
意外之财、横财
流动性资产
```

**性格特征**：
```
大方、豪爽
爱投机、风险意识弱
```

**特点**：
```
偏财旺 → 收入来源多
偏财克偏印 → 艺术天赋
偏财透干 → 异性缘好
```

### 7. 正官

**定义**：克日干、同阴阳

**象征**：
```
丈夫（女命正官为夫）
上司、政府
名誉、地位
```

**性格特征**：
```
正直、负责、有原则
保守、缺乏变通
```

**特点**：
```
正官旺 → 贵气足
正官生正印 → 贵
官星被克 → 事业受阻
```

### 8. 七杀

**定义**：克日干、不同阴阳

**象征**：
```
小人、敌人
压力、危机
权力、武职
```

**性格特征**：
```
果断、魄力、执行力强
霸道、易怒、极端
```

**特点**：
```
七杀攻身 → 健康问题
七杀有制 → 化杀为权
七杀无制 → 祸患无穷
```

### 9. 正印

**定义**：生日干、同阴阳

**象征**：
```
母亲、长辈
学历、证书
房产、靠山
```

**性格特征**：
```
善良、有爱心
依赖、缺乏主见
```

**特点**：
```
正印旺 → 学业好
正印生身 → 有靠山
印星克食神 → 保守
```

### 10. 偏印

**定义**：生日干、不同阴阳

**象征**：
```
继母、长辈
非正统学业
玄学、偏门
```

**性格特征**：
```
敏感、悟性高
孤僻、心机重
```

**特点**：
```
偏印克食神 → 子女缘薄
偏印有制 → 特殊才能
偏印旺 → 适合钻研
```

## 十神组合分析

### 财富组合
```
食神生财 → 财源滚滚
财星配印 → 财权双收
身旺财旺 → 富命
身弱财多 → 富屋贫人
```

### 事业组合
```
官星配印 → 权贵双全
七杀有制 → 权威人物
伤官配印 → 学术权威
```

### 婚姻组合
```
男命正财旺 → 婚姻稳定
女命官星旺 → 婚姻幸福
财星克印 → 婆媳问题
```

### 性格组合
```
印星旺 → 温和、保守
食伤旺 → 活泼、创新
官杀旺 → 正直、有原则
比劫旺 → 好胜、竞争
```

## 十神实战分析

### 案例1：甲寅 丙寅 戊辰 丙辰

```
日主：戊土
分析：
- 年柱甲寅：甲=偏财，寅=比肩+食神+偏印
- 月柱丙寅：丙=食神，寅=比肩+食神+偏印
- 日柱戊辰：戊=比肩，辰=戊+癸+乙
- 时柱丙辰：丙=食神，辰=戊+癸+乙

十神分布：
- 财星：甲（偏财）×2
- 官杀：无
- 印星：寅中偏印×2，辰中乙=正印
- 食伤：丙×2（食神）+ 寅中食神×2
- 比劫：戊×2（比肩）+ 辰中戊×2
```

## 练习题

### 练习1：标出十神
```
八字：癸卯 乙卯 丙子 辛卯

日主：丙火
分析：
- 癸卯 → 癸=正官，卯=偏印
- 乙卯 → 乙=正印，卯=偏印
- 丙子 → 丙=比肩，子=七杀
- 辛卯 → 辛=正财，卯=偏印

答案：
癸卯：正官、偏印
乙卯：正印、偏印
丙子：比肩、七杀
辛卯：正财、偏印
```

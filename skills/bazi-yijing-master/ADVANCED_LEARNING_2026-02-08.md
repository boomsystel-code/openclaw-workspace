### 大运流年关系

#### 命局 vs 大运 vs 流年
```
命局：先天禀赋，终身不变
大运：后天运势，10年一段
流年：当年变化，1年一变

关系：
命局 × 大运 → 整体走势
命局 × 大运 × 流年 → 具体应期
```

#### 运程判断
```
用神到位：好运
```

## ❓ 进阶疑问与解答

### Q1：身弱但财旺怎么办？

A1：
```
分析：
1. 财旺不一定是好事
2. 身弱担不住财
3. 财多反而是压力

解决：
1. 先用印星生身
2. 再看财星可否担起
3. 建议稳健理财
```

### Q2：官杀混杂怎么办？

A2：
```
分析：
1. 官杀混杂事业波折
2. 需有制化
3. 食伤制杀为上

建议：
1. 走技术路线
2. 避免官场
3. 用专业说话
```

### Q3：大运逢冲一定是坏事？

A3：
```
不一定：
1. 冲用神：坏事
2. 冲忌神：好事
3. 冲婚姻宫：感情变动
4. 冲事业宫：事业变动

需具体分析：
- 冲的性质（天克地冲/地支相冲）
- 冲的结果（用神得失）
- 冲的时机（应期）
```

### Q4：神煞越多越好吗？

A4：
```
不完全是：
1. 吉神多且旺：好
2. 吉神受克：无力
3. 凶神多：有制化可化解
4. 凶神无制：坏事

关键：
神煞状态比数量更重要
```

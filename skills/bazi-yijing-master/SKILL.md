## 第一阶段：基础知识（2周）

### 📅 Day 1-3: 八字基础概念

#### 1.1 什么是八字
```
八字 = 年柱 + 月柱 + 日柱 + 时柱
      (年干年支) (月干月支) (日干日支) (时干时支)

每个柱 = 1个天干 + 1个地支 = 2个字
共4柱 × 2字 = 8字
```

#### 1.4 五行相生相克

**相生**（木→火→土→金→水→木）
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
```

**相克**（木克土→土克水→水克火→火克金→金克木）
```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```

### 📅 Day 4-7: 五行与阴阳

#### 2.1 五行特性
| 五行 | 特性 | 象征 | 对应方位 | 对应季节 |
|------|------|------|----------|----------|
| **木** | 生长、条达 | 仁 | 东 | 春 |
| **火** | 炎热、向上 | 礼 | 南 | 夏 |
| **土** | 生化、承载 | 信 | 中 | 四季 |
| **金** | 清洁、收敛 | 义 | 西 | 秋 |
| **水** | 寒冷、向下 | 智 | 北 | 冬 |

#### 2.2 阴阳概念
```
阳 = 动、明、外、上、热、积极
阴 = 静、暗、内、下、寒、消极

天干：甲丙戊庚壬 = 阳
      乙丁己辛癸 = 阴

地支：子寅辰午申戌 = 阳
      丑卯巳未酉亥 = 阴
```

### 📅 Day 8-10: 排八字

#### 3.1 年柱
- 以立春为界
- 查万年历或用排盘软件

#### 3.2 月柱
- 以节令为界（24节气中的节）
- 月干推算口诀：
```
甲己之年丙作首
乙庚之年戊为头
丙辛必定寻庚起
丁壬壬位顺行流
戊癸之年何方发
甲寅之上好追求
```

#### 3.3 日柱
- 需查万年历
- 或用公式计算

#### 3.4 时柱
- 以子时为界（23:00-01:00）
- 时干推算：
```
甲己还生甲
乙庚丙作初
丙辛从戊起
丁壬庚子居
戊癸何方发
壬子是真途
```

### 📅 Day 11-14: 五行分析

#### 4.1 日主（身强身弱）
- 日干 = 日主 = 自己
- 计算日主五行得令、得地、得势情况
- 判断身强/身弱

#### 4.2 十神
```
以日干为中心：
生我者 → 正印、偏印
我生者 → 食神、正财
克我者 → 正官、七杀
我克者 → 正财、偏财
同我者 → 比肩、劫财
```

#### 4.3 用神与忌神
- **用神**：对日主有利的五行
- **忌神**：对日主有害的五行

## 💡 学习心得

### 初学者建议
1. **循序渐进** - 不要急，从基础开始
2. **多练多算** - 每天至少排1-2个八字
3. **交叉验证** - 用多个软件核对结果
4. **名师指点** - 有条件找老师傅校正

### 常见误区
- ❌ 死记硬背 - 要理解原理
- ❌ 过度依赖神煞 - 只能辅助
- ❌ 预测结果绝对化 - 要留有余地
- ❌ 只学理论不实践 - 必须实操

# 易经与梅花易数入门

## 易经基础

### 易经核心
```
易 = 变化
经 = 经典

易经 = 讲述变化规律的经典
```

### 太极
```
无极 → 太极 → 两仪 → 四象 → 八卦 → 六十四卦

太极 = 一 = 混沌未分的状态
两仪 = 阴阳
四象 = 太阳、少阴、太阴、少阳
八卦 = 三爻组合
六十四卦 = 六爻组合
```

## 阴阳

### 阴阳概念
```
阳 = 动、明、外、上、热、积极
阴 = 静、暗、内、下、寒、消极
```

### 阴阳特性
```
对立：日/夜，男/女，高/低
统一：孤阴不生，独阳不长
转化：物极必反，否极泰来
消长：此消彼长，动态平衡
```

### 阴阳符号
```
阳爻 ──（实线）
阴爻 -- --（虚线）
```

### 卦象结构
```
上卦（外卦）← 三爻
下卦（内卦）← 三爻

例：泰卦 = 地上水下 = 地天泰
上坤（地）
下乾（天）
```

### 卦序
```
1-8：上经（天地到水火）
9-64：下经（咸恒到未济）
```

## 梅花易数

### 起卦方法

#### 1. 时间起卦
```
年基数 + 月基数 + 日基数 = 上卦
上卦 + 时基数 = 下卦
（上卦+下卦+时辰数）÷ 8 = 余数为动爻
```

**年基数：**
```
子=1 丑=2 寅=3 卯=4 辰=5 巳=6
午=7 未=8 申=9 酉=10 戌=11 亥=12
```

**月基数 = 月份**

**日基数 = 日期**

**时基数：**
```
子=1 丑=2 寅=3 卯=4 辰=5 巳=6
午=7 未=8 申=9 酉=10 戌=11 亥=12
```

**时辰 = 23:00-01:00 = 1，01:00-03:00 = 2 ...**

#### 2. 数字起卦
```
任意报3个数：A B C
上卦 = A ÷ 8 余数
下卦 = B ÷ 8 余数
动爻 = C ÷ 6 余数
```

**余数对应：**
```
1=乾 2=兑 3=离 4=震 5=巽 6=坎 7=艮 8=坤
```

## 解卦基础

### 1. 主卦
```
起卦得到的六爻卦
代表当前状态
```

### 2. 变卦
```
动爻阴阳互换后得到的卦
代表最终结果
```

### 3. 互卦
```
主卦中间四爻重新组合
代表过程、细节
```

### 4. 体用分析
```
体卦 = 问事人/主卦下卦
用卦 = 问事/主卦上卦

体生用：泄气（不利）
用生体：进气（有利）
体克用：可成（费力）
用克体：不成（有害）
体用比和：顺利
```

### 5. 动静分析
```
静卦：无动爻 → 事情暂时静止
有动爻：事情在变化中
```

## 实战案例

### 案例1：时间起卦

```
时间：2026年2月8日10:00（辰时）
年：辰=5
月：2
日：8
时：辰=5

上卦：(5+2+8)÷8=15÷8=7...余7 → 艮
下卦：(7+5)÷8=12÷8=4...余4 → 震
动爻：(7+5+5)÷6=17÷6=2...余5 → 5爻动

主卦：火山旅（☲☶）
变卦：山天大畜（☴☰）
动爻：5爻
```

**分析：**
```
体卦：艮土
用卦：离火
体生用 → 泄气

体用比和 → 中平
```

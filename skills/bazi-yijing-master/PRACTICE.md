## 二、财富分析

### 2.1 财星判断

```
正财：正当收入、工资、妻子（男命）
偏财：意外之财、投资、父亲（男命）
```

### 2.2 财星分析要点

#### 1. 财星旺衰
```
财星旺：收入高、财运好
财星弱：收入一般、需努力
财星多：财源广
财星少：财源有限
```

#### 2. 身财平衡
```
身旺财旺：富命
身弱财多：富屋贫人（有钱享受不了）
身旺财弱：财来财去
身弱财弱：清贫
```

#### 3. 财星位置
```
月令财：得令，财运好
时支财：晚年财
日支财：直接相关
年柱财：祖产
```

### 2.3 案例分析

#### 案例1：戊子 己未 丙寅 戊戌

```
财星分析：
- 财星：己（正财）×2，戊（偏财）×2
- 财星旺度：偏旺

日主：丙火
身旺度：偏弱

身财关系：
- 身弱不担财
- 财多但享受有限
- 需印星生扶

建议：
- 提升自身能力
- 财不外露
- 稳健投资
```

## 三、事业分析

### 3.1 官杀判断

```
正官：正当权力、贵人、丈夫（女命）
七杀：权力、压力、小人、武职
```

### 3.2 官杀分析要点

#### 1. 官杀旺衰
```
官杀旺：事业心强、压力大大
官杀弱：事业心不足
官杀纯：仕途顺利
官杀杂：事业波折
```

#### 2. 身官平衡
```
身旺官旺：官贵命
身弱官旺：压力山大
身旺官弱：怀才不遇
身官相停：最佳状态
```

#### 3. 官杀组合
```
官印相生：权印双收
杀印相生：武贵
官杀混杂：事业波折
伤官见官：事业受阻
```

### 3.3 案例分析

#### 案例2：甲寅 丙寅 戊辰 丙辰

```
官杀分析：
- 官杀：无（七杀、正官）
- 官杀状态：缺失

日主：戊土
身旺度：偏弱

事业分析：
- 官星不现
- 事业运一般
- 适合技术路线
- 不适合官场

建议：
- 走技术路线
- 避免官场
- 创业或自由职业
```

## 四、感情分析

### 4.1 婚姻星判断

```
男命：正财为正妻，偏财为桃花
女命：正官为正夫，七杀为桃花
```

### 4.2 感情分析要点

#### 1. 婚姻星状态
```
婚姻星旺：感情丰富
婚姻星弱：感情平淡
婚姻星纯：专一
婚姻星杂：感情复杂
```

#### 2. 日支分析
```
日支为喜用：婚姻幸福
日支为忌神：婚姻需注意
日支逢冲：婚姻变动
日支逢合：婚姻稳定
```

| 丑 | 稳重、务实 | 属蛇、鸡、牛 |
| 寅 | 热情、上进 | 属马、狗、羊 |
| 卯 | 温柔、善良 | 属猪、羊、狗 |
| 辰 | 包容、诚信 | 属鸡、牛、龙 |
| 巳 | 聪明、敏感 | 属猴、蛇、鸡 |
| 午 | 热情、浪漫 | 属虎、狗、羊 |

### 4.3 案例分析

#### 案例3：丙子 壬辰 乙酉 庚辰

```
感情分析：
- 婚姻星：壬（偏印）、庚（正官）
- 官星：庚（正官）
- 日支：酉

日主：乙木
身旺度：偏弱

感情特点：
- 正官为用，官星有力
- 日支酉金为忌神
- 婚姻需注意

建议：
- 晚婚为宜
- 注意婆媳关系
- 找温和对象
```

## 五、健康分析

### 5.1 五行与脏腑

```
木 → 肝胆
火 → 心小肠
土 → 脾胃
金 → 肺大肠
水 → 肾膀胱
```

### 5.2 健康分析要点

#### 1. 五行缺失
```
某五行弱或缺失 → 对应脏腑需注意
某五行过旺 → 对应脏腑过旺
```

#### 2. 日主状态
```
身弱：整体抵抗力弱
身旺：某些功能过旺
```

#### 3. 特定组合
```
水弱：肾虚、腰不好
木弱：肝胆问题
火弱：心脏问题
土弱：脾胃问题
金弱：肺问题
```

### 5.3 案例分析

#### 案例4：甲寅 丙寅 戊辰 丙辰

```
健康分析：
- 日主：戊土
- 土的状态：偏弱

脏腑关注：
- 脾胃：重点关注
- 消化系统需注意

预防建议：
- 规律饮食
- 避免生冷
- 适当运动
- 定期体检
```

| 伤官 | 创新、叛逆、口才好 |
| 比肩 | 独立、好胜、人缘差 |
| 劫财 | 机敏、投机、夺财 |

### 6.2 综合性格分析

#### 案例5：甲寅 丙寅 戊辰 丙辰

```
十神分布：
- 印星：丙×2（食神）×2
- 比劫：戊×2
- 财星：甲×2

性格特点：
- 印星旺：温和、依赖、学习型
- 比劫旺：独立、竞争
- 财星中：务实

综合：
- 外向开朗
- 学习能力强
- 有独立思考
- 适合团队协作
- 理财需注意
```

## 九、学习方法

### 9.1 分析技巧

```
1. 先整体后局部
2. 先旺衰后喜忌
3. 先十神后神煞
4. 先命局后大运
```

### 9.2 常见错误

```
1. 只看一点不全面
2. 死套公式不灵活
3. 忽略组合关系
4. 过于依赖神煞
```

### 9.3 提升方法

```
1. 多看古籍经典
2. 多分析实际案例
3. 多请教师傅校正
4. 多实践积累经验
```

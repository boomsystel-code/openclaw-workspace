# 大运流年深度解析

## 一、大运基础

### 1.1 大运概念

大运 = 将人生划分为每10年一个阶段的运势曲线

```
特点：
- 以月柱为基准
- 阳男阴女顺排
- 阴男阳女逆排
- 每步运管10年
```

### 1.2 起运时间计算

#### 规则
```
出生到下一个节气的天数 ÷ 3 = 起运岁数

不足3天 → 按3天算
超过3天不足6天 → 按6天算
以此类推
```

#### 例子
```
案例：2024年4月10日出生
下一个节气：立夏（5月5日）
间隔天数：25天
起运岁数：25 ÷ 3 ≈ 8岁
```

### 1.3 顺逆判断

#### 口诀
```
阳男阴女顺
阴男阳女逆

阳年：甲丙戊庚壬
阴年：乙丁己辛癸
```

#### 判断流程
```
1. 确定年干阴阳
   - 甲丙戊庚壬 = 阳
   - 乙丁己辛癸 = 阴
   
2. 确定性别
   - 男 = 阳
   - 女 = 阴
   
3. 判断顺逆
   - 阳男/阴女 → 顺排
   - 阴男/阳女 → 逆排
```

### 1.4 大运排列

#### 顺排规则
```
月柱 → 年柱方向 → 年柱 → 月柱 → 日柱 → 时柱
（年→月→日→时，依次后退）
```

#### 逆排规则
```
月柱 → 时柱方向 → 时柱 → 日柱 → 月柱 → 年柱
（时→日→月→年，依次前进）
```

#### 例子：顺排
```
月柱：己巳
顺排：庚午 → 辛未 → 壬申 → 癸酉 → 甲戌 → 乙亥 → 丙子 → 丁丑 → 戊寅 → 己卯
```

#### 例子：逆排
```
月柱：己巳
逆排：戊辰 → 丁卯 → 丙寅 → 乙丑 → 甲子 → 癸亥 → 壬戌 → 辛酉 → 庚申 → 己未
```

## 二、流年基础

### 2.1 流年概念

流年 = 当年的干支
流月 = 当月的干支
流日 = 当日的干支
流时 = 当时辰的干支

```
2024年 → 甲辰年
2024年3月 → 丙辰月（农历二月）
```

### 2.2 流年干支速查

#### 年干支对照表
```
2024：甲辰  2034：甲寅
2025：乙巳  2035：乙卯
2026：丙午  2036：丙辰
2027：丁未  2037：丁巳
2028：戊申  2038：戊午
2029：己酉  2039：己未
2030：庚戌  2040：庚申
2031：辛亥  2041：辛酉
2032：壬子  2042：壬戌
2033：癸丑  2043：癸亥
```

#### 月干支速查
```
年干基数 + 月份 = 流月干
（配合月干表使用）
```

## 三、命局+大运+流年分析

### 3.1 三者关系

```
命局（固定）← 大运（10年一段）← 流年（1年一变）
```

#### 命局
- 先天禀赋
- 终身不变
- 决定基本格局

#### 大运
- 后天运势
- 10年一换
- 影响整体走势

#### 流年
- 当年变化
- 1年一换
- 影响具体应期

### 3.2 分析框架

#### 第一步：分析命局
```
1. 排出八字
2. 判断日主强弱
3. 确定用神忌神
4. 分析格局高低
5. 找出关键十神
```

#### 第二步：分析大运
```
1. 大运干支五行
2. 与命局关系（生克合冲）
3. 整体走势（好运/坏运）
4. 重点应期（哪些年要特别注意）
```

#### 第三步：分析流年
```
1. 流年干支五行
2. 与大运关系
3. 与命局关系
4. 具体应事（哪方面应验）
```

### 3.3 组合分析

#### 用神到位（好运）
```
用神在流年出现
用神被生扶
忌神被克制
```

#### 忌神当道（坏运）
```
忌神在流年出现
忌神被生扶
用神被克制
```

#### 特殊组合
```
天克地冲：重大变化
岁运并临：应期强烈
合入命局：好事将近
冲克命局：坏事将至
```

## 四、实战分析案例

### 案例1：甲寅 丙寅 戊辰 丙辰

#### 1. 命局分析
```
日主：戊土
月令：寅木（春季）
日主状态：失令、失地、偏弱

用神：火、土（生扶日主）
忌神：水、金、木（克泄耗日主）
```

#### 2. 大运分析（假设）
```
庚午 辛未 壬申 癸酉 甲戌 乙亥 丙子 丁丑

第一步运（庚午）：
- 庚=七杀，午=正印
- 午火生戊土 → 有利
- 庚克甲木 → 有利
- 整体：中等偏上
```

#### 3. 流年分析（2024甲辰）
```
流年：甲辰
- 甲=偏财，辰=比肩+戊+癸+乙
- 甲木克戊土 → 辰土有根
- 甲木生丙火 → 间接生土
- 辰土帮身 → 有利
- 2024年整体：平顺
```

## 六、进阶技巧

### 6.1 细批流年

#### 流年四柱
```
年柱：宏观环境
月柱：当月重点
日柱：核心变化
时柱：年末收尾
```

#### 分析要点
```
1. 干支五行属性
2. 与命局关系
3. 与大运关系
4. 十神类象
5. 神煞配合
```

### 6.2 应期判断

#### 远近
```
远：慢，应在后期
近：快，应在初期
```

#### 强弱
```
强：明显，应事大
弱：轻微，应事小
```

#### 生克
```
生：正面影响
克：负面影响
冲：剧烈变化
合：稳定持续
```

## 八、学习心得

### 8.1 核心理念
```
1. 命局是根本，大运是框架，流年是细节
2. 顺势而为，逆势而守
3. 好运不骄，坏运不馁
4. 命越算越薄（不可过度依赖）
```

### 8.2 常见误区
```
1. 迷信：完全相信宿命论
2. 宿命：认为一切都是注定的
3. 过度：频繁算命、患得患失
4. 绝对：认为100%准确
```

### 8.3 正确态度
```
1. 参考：作为人生参考
2. 调整：根据运势调整策略
3. 努力：运势只是加分项
4. 心态：尽人事听天命
```

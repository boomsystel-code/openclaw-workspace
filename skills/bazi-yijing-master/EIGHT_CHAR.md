# 八字排盘完整教程

## 排盘基础

### 八字结构
```
年柱  月柱  日柱  时柱
  ↓     ↓     ↓     ↓
年干   月干   日干  时干
年支   月支   日支  时支

例：甲寅  丙寅  戊辰  丙辰
    甲    丙    戊    丙
    寅    寅    辰    辰
```

### 排盘四要素
1. 出生年月日时（阳历/公历）
2. 出生地点（影响真太阳时）
3. 性别（男命女命有别）
4. 出生时刻（精确到时辰）

## 第二步：年柱

### 规则
- 以**立春**为分界线
- 立春前出生 → 上一年
- 立春后出生 → 当年

### 示例
```
2024年2月4日12:00前出生 → 癸卯年
2024年2月4日12:00后出生 → 甲辰年
```

### 万年历查询
```
年份 → 干支对照：
2024 → 甲辰
2023 → 癸卯
2022 → 壬寅
2021 → 辛丑
2020 → 庚子
```

### 规则
- 以**节气**为分界线
- 月令 = 月支 = 出生月份所在节气

秋季：立秋 处暑 白露 秋分 寒露 霜降
冬季：立冬 小雪 大雪 冬至 小寒 大寒

### 月干推算（核心口诀）

**五虎遁年起月干：**
```
甲己之年丙作首
乙庚之年戊为头
丙辛必定寻庚起
丁壬壬位顺行流
戊癸之年何方发
甲寅之上好追求
```

**解释：**
```
甲年、己年 → 正月=丙寅
乙年、庚年 → 正月=戊寅
丙年、辛年 → 正月=庚寅
丁年、壬年 → 正月=壬寅
戊年、癸年 → 正月=甲寅
```

**月干表：**
```
            甲 乙 丙 丁 戊 己 庚 辛 壬 癸
            ↓  ↓  ↓  ↓  ↓  ↓  ↓  ↓  ↓  ↓
甲年/己年 → 丙 丁 戊 己 庚 辛 壬 癸 甲 乙
乙年/庚年 → 戊 己 庚 辛 壬 癸 甲 乙 丙 丁
丙年/辛年 → 庚 辛 壬 癸 甲 乙 丙 丁 戊 己
丁年/壬年 → 壬 癸 甲 乙 丙 丁 戊 己 庚 辛
戊年/癸年 → 甲 乙 丙 丁 戊 己 庚 辛 壬 癸
```

### 快速记忆表
```
       正月→二月→三月→四月→五月→六月
甲己  丙    丁    戊    己    庚    辛
乙庚  戊    己    庚    辛    壬    癸
丙辛  庚    辛    壬    癸    甲    乙
丁壬  壬    癸    甲    乙    丙    丁
戊癸  甲    乙    丙    丁    戊    己

       七月→八月→九月→十月→冬月→腊月
甲己  壬    癸    甲    乙    丙    丁
乙庚  甲    乙    丙    丁    戊    己
丙辛  丙    丁    戊    己    庚    辛
丁壬  戊    己    庚    辛    壬    癸
戊癸  庚    辛    壬    癸    甲    乙
```

### 例子
```
假设：2024年4月10日出生

1. 年柱：2024年 → 甲辰年
2. 月份：4月 → 清明后、谷雨前 → 辰月
3. 月干：甲年 → 甲年首月是丙寅
   丙→丁→戊→己→庚（4月=己巳）
   
结果：月柱 = 甲辰 己巳
```

## 第四步：日柱

### 规则
- 以23:00（子时）为界
- 23:00前 → 当日
- 23:00后 → 次日

### 计算方法
日柱需要查万年历或用公式计算。

**公式法（简化）：**
```
已知：2024年2月10日为甲子日
每天减1，干支循环60

示例：计算2024年4月10日
从2月10日→4月10日 = 60天
60÷60=1组，干支相同
结果：甲子日
```

**推荐直接查万年历！**

### 时干推算（核心口诀）

**五鼠遁日起时干：**
```
甲己还生甲
乙庚丙作初
丙辛从戊起
丁壬庚子居
戊癸何方发
壬子是真途
```

**解释：**
```
甲日、己日子时 = 甲干
乙日、庚日子时 = 丙干
丙日、辛日子时 = 戊干
丁日、壬日子时 = 庚干
戊日、癸日子时 = 壬干
```

**时干表：**
```
       子  丑  寅  卯  辰  巳  午  未  申  酉  戌  亥
       ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓   ↓
甲己  甲   乙   丙   丁   戊   己   庚   辛   壬   癸   甲   乙
乙庚  丙   丁   戊   己   庚   辛   壬   癸   甲   乙   丙   丁
丙辛  戊   己   庚   辛   壬   癸   甲   乙   丙   丁   戊   己
丁壬  庚   辛   壬   癸   甲   乙   丙   丁   戊   己   庚   辛
戊癸  壬   癸   甲   乙   丙   丁   戊   己   庚   辛   壬   癸
```

### 例子
```
假设：2024年4月10日 08:30 出生

1. 日柱：甲子日（已查万年历）
2. 时辰：07:00-09:00 → 辰时
3. 日干：甲
4. 甲日子时 = 甲
   甲→乙→丙→丁→戊（辰时=戊辰）
   
结果：时柱 = 戊辰
```

## 大运排法

### 规则
- 顺逆判断：
  - 阳年（甲丙戊庚壬）→ 顺排
  - 阴年（乙丁己辛癸）→ 逆排

### 起运时间
```
出生到下一个节气的天数 ÷ 3 = 起运岁数

例：4月10日出生，下一个节是立夏（5月5日）
    间隔25天
    起运：25 ÷ 3 ≈ 8岁
```

### 排运表
```
顺排：年→月→年→月→...
逆排：年→月→年→月→...
```

## 练习题

### 练习1：排出1990年8月15日15:30的八字
```
步骤：
1. 年柱：1990年 → 庚午
2. 月柱：8月 → 立秋后 → 申月
   庚年首月是丙寅
   丙→丁→戊→己→庚→辛→壬→癸（8月=壬申）
3. 日柱：查万年历 → 丙戌日
4. 时柱：15:30 → 申时
   丙日子时 = 戊
   戊→己→庚→辛（申时=辛申）
5. 大运：庚年=阳 → 顺排

答案：
年柱：庚午
月柱：壬申
日柱：丙戌
时柱：辛申
```

## 常见错误

### ❌ 错误1：弄错年柱分界
**正确**：以立春为界，非春节

### ❌ 错误2：月干推算错误
**正确**：熟记口诀，多加练习

### ❌ 错误3：时柱用错
**正确**：以23:00为子时起点

### ❌ 错误4：忽略真太阳时
**正确**：出生地与北京时间有差异

## 🎯 投资策略

### 技术分析
- RSI > 70 → 超买 → 可能下跌
- RSI < 30 → 超卖 → 可能上涨
- MACD金叉 → 上涨信号
- MACD死叉 → 下跌信号

### 市场情绪
| 恐惧贪婪指数 | 含义 |
|-------------|------|
| 0-25 | 极度恐惧 → 可能买入 |
| 25-50 | 恐惧 |  
| 50-75 | 贪婪 | 75-100 | 极度贪婪 → 可能卖出 |

### 综合决策
```
技术面 + 情绪面 + AI预测
    ↓
  多数看涨 → 买入
  多数看跌 → 卖出
  不一致 → 观望
```
